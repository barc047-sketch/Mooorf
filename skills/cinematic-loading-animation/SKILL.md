Use for the ZONUERT intro loader.

- full-screen overlay
- corner countdown
- red timer/link text
- colorful liquid gradient covering ~30% screen
- GSAP timeline preferred
- smooth reveal into canvas
- premium editorial tone
- not childish/rainbow/gaming

Sequence:
1. Base fades in.
2. Gradient mass blooms.
3. Countdown runs.
4. Status labels update.
5. Canvas previews underneath.
6. Loader exits.
7. UI docks reveal.
