# General Agent Handoff

Before work:

1. Read `docs/HANDOFF.md`.
2. Read `docs/TASK_QUEUE.md`.
3. Confirm the requested phase and protected core.

At handoff, record:

- phase and completion status;
- files changed;
- build and preview evidence;
- store/canvas/table reset status when relevant;
- real bugs and deferred work;
- exact next phase, without starting it.

Do not repeat the full project history. Link to existing policy documents rather than duplicating them.
