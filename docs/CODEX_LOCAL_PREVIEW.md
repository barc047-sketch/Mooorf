# Codex Local Preview

Run:

```sh
npm run dev
```

Expected URL: `http://localhost:5173`, unless Vite reports another port.

Build:

```sh
npm run build
```

Verify:

- the app renders without a blank screen;
- the canvas is visible;
- Canvas/Table switching works without reset;
- the blob toggle works;
- there are no console-breaking errors.

Do not deploy the preview online.
