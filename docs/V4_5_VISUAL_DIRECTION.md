- **Superpower** → soft grey-white glass metric widgets, compact technical typography, animated key numbers.

## Final direction
Palmer editorial canvas + Rayon CAD usability + Superpower metric glass + dark scan analysis UI + Graph Noir Red identity.

## Base rules
- Grayscale-first UI: cream / white / grey / black.
- High-contrast black/grey mode supported alongside day/night.
- Cream editorial day mode preserved (existing token system).
- Graph Noir Red remains the night-mode identity.
- Deep wine red reserved for identity, active states, warnings, highlights — not decorative.
- Muted yellow reserved for warning/alert states only.
- Soft gradients confined to metric widgets, active-tool accents, organism category blend.
- Dotted technical canvas background (optional 1m×1m grid scale, see [V4_5_CANVAS_GRID_AND_SCALE_SYSTEM.md](V4_5_CANVAS_GRID_AND_SCALE_SYSTEM.md)).
- Glassmorphism widgets — premium rounded panels, not cheap blur spam.
- Compact technical labels throughout (CAD-style, not marketing copy).
- Floating widgets sit over the canvas, not docked into a boring dashboard page.

## Explicitly avoid
- Random neon colors.
- Childish or bouncy animation.
- Generic SaaS cards.
- Boring flat dashboard pages.
- Copying Palmer branding or any proprietary reference asset.

## Preserve
- ZONUERT spatial-graph identity: central graph drives canvas + table + future widgets, always.
