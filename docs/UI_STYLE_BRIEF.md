# UI Style Brief

## Day mode
- Gallery Cream: #f5f6ee
- Ink: #222222
- Fog: #a1a19c
- Bone: #ffffff
- Lampblack: #000000
- ZONUERT red as link/timer/accent only

## Night mode
- Near black: #070707
- Charcoal: #161616
- Deep red: #8f1111
- Blood red: #c31616
- Oxblood: #4a0909
- Fog grey: #9a9a94

## Typography
Use Inter Tight / Helvetica-like / Neue Haas style substitute.

Rules:
- tight negative tracking
- small UI text
- giant editorial text only in loader/empty states
- no chunky SaaS headings

## Layout
- full viewport
- no canvas page scroll
- top-left ZONUERT mark
- top-center canvas/table pill
- bottom-center dock
- left vertical rail
- bottom-right zoom
- loader countdown in corner

## Controls
- 1px hairline border
- rounded pills for toggles
- 3px/9px/100px radius system
- glass blur allowed for ZONUERT controls
- no heavy drop shadows on UI
- cells can have object shadows

## Final visual target
Palmer editorial object canvas + iOS glassmorphism controls + Graph Noir Red architecture tool + premium animated agency-level interface.

## Must avoid
- generic SaaS UI
- random neon
- childish animation
- pipe bridges (organism/blob)
- red halo spam
