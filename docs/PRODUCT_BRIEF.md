# Product Brief

ZONUERT is a spatial program visualization studio for architecture students/designers.

It transforms:
- area schedules
- space programs
- relationship data

into:
- editorial spatial diagrams
- bubble/cell canvases
- morph/organism zoning
- table-linked diagrams
- future exports/presentation sheets

This lab proves:
- premium animated editorial canvas
- table/canvas sync
- organism renderer
- Palmer-level interaction quality
